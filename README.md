# Fantasy Leagues

Research, scoring analysis, and auction strategy for dynasty fantasy football leagues.

Each directory is one league. Files inside:
- `league.json` — Sleeper IDs, roster settings, scoring rules
- `auction-budget.md` — positional budget and target ceilings
- `player-values-YYYY-YYYY.md` — scored stats from actual seasons using that league's rules
